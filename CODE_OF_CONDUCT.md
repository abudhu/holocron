# Code of Conduct

This project follows the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct.html).

The short version: **be respectful, assume good faith, focus on the
code rather than the person**. Technical disagreement is welcome and
expected; personal attacks, harassment, discriminatory language, and
sustained disruption are not.

## Scope

This applies to all project spaces:

- The OneDev repository (<https://onedev.amitbudhu.com/holocron>) — issues,
  PRs, code review comments
- Any GitHub mirror's issue tracker / pull request discussions
- Any reddit / discord / matrix room where the project is officially
  represented
- One-on-one communications with maintainers when the contact is
  project-related

## Enforcement

If you experience or witness behavior that violates this Code of
Conduct, please report it to **conduct@amitbudhu.com**.

All reports will be:

- Reviewed within 72 hours
- Kept confidential to the extent possible while still investigating
- Acted on with the response level appropriate to the violation (from
  a private warning to a permanent ban from project spaces)

The maintainer (currently
[Amit Budhu](https://github.com/abudhu)) is the final arbiter of
enforcement decisions. Decisions can be appealed once, in writing, by
replying to the original enforcement message.
